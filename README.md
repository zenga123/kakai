# 가까이

![가까이 앱 이미지 일본어](https://github.com/user-attachments/assets/3088b70d-1799-42e7-9a1e-202af9d6330e)

「がっかい」はカップル向けのiOSアプリで、恋人と過ごした時間を視覚的に確認しながら、  
次のデート日程までを簡単に管理できる機能を提供します。

## :hammer_and_wrench: 技術スタック

### iOS (Swift)
- **SwiftUI**  
  - 宣言型UIを用いたモダンな開発
- **MVVMアーキテクチャ**  
  - データとUIの分離を実現
- **WidgetKit**  
  - ホーム画面ウィジェットの実装
- **App Groups**  
  - アプリとウィジェット間でのデータ共有
- **UserDefaults**  
  - 軽量なデータ永続化に利用
- **PhotosUI**  
  - 写真選択機能の実装
